# Logic-Lens Outreach Roadmap

**Goal**: grow from 2 GitHub stars + 1 solo maintainer to a community of users, contributors, and third-party marketplace listings. Target: 100 stars and inclusion in 3+ Claude plugin marketplaces by end of Q2.

## Layer 1 — Repo infrastructure (done in v0.4.3)

- [x] GitHub topics (claude-code, claude-plugin, code-review, etc.) for SEO
- [x] Branch protection on `main` (no force-push, no deletion, PR required)
- [x] Discussions enabled
- [x] SECURITY.md + CODE_OF_CONDUCT.md + issue templates + PR template
- [x] .github/FUNDING.yml placeholder
- [x] README badges (stars, license, plugin manifests, verification score)

## Layer 2 — Claude-ecosystem marketplace submissions (next action)

Research + submit Logic-Lens to third-party Claude plugin marketplaces. Most accept PRs adding an entry to their `marketplace.json`.

### Known marketplaces to submit to

| Marketplace | Submission method | Priority |
|---|---|---|
| `hesreallyhim/awesome-claude-code` | PR add entry to README table | P0 (highest traffic) |
| `davila7/claude-code-plugin-marketplace` | PR add entry to marketplace.json | P0 |
| `claude-plugins` community hubs | various — TBD | P1 |
| Anthropic's official plugin directory | Wait for public submission flow (currently invite-only) | P2 |

Template PR body: see `docs/announcement/marketplace-submission.md`.

### Steps for each marketplace PR

1. Fork the marketplace repo.
2. Add the Logic-Lens entry following that repo's schema.
3. Include a short description, link to logic-lens repo, and v0.4.x release URL.
4. Reference our `/logic-review` `/logic-explain` `/logic-diff` `/logic-locate` `/logic-health` `/logic-fix-all` commands.
5. Open PR. Tag maintainer in description.

## Layer 3 — Technical community channels

### Reddit
- `r/ClaudeAI` — Show HN style post. Best time: Tue–Thu 9am ET. Draft: `docs/announcement/reddit-claudeai.md`.
- `r/LocalLLaMA` — cross-post focused on tooling angle.
- `r/programming` — lead with the "semi-formal tracing vs lint" angle, not Claude-specific.

### Hacker News
- Title: `Show HN: Logic-Lens — Find logic bugs linters miss via semi-formal execution tracing`
- Post on a Tuesday 9am PT for max visibility. Draft: `docs/announcement/hn-show-hn.md`.
- Be present in comments for at least 4 hours afterwards.

### Dev.to / Hashnode
- Long-form article with v0.4.0 case study. 1,500-2,500 words with code examples.
- Draft: `docs/announcement/devto-blog.md`.

### Twitter/X
- Thread of 6–8 tweets. Draft: `docs/announcement/twitter-thread.md`.
- Tag @AnthropicAI and @claude_code.

### Anthropic Discord
- Post in `#showcase` or similar channel.
- Share the GitHub release link + one concrete example.

### LinkedIn
- Short post focused on the "what it finds that linters don't" angle.
- Target the engineering-leadership audience.

## Layer 4 — Content marketing (long-term, weekly cadence)

- **Case studies**: use `/logic-health` on popular open-source repos, publish findings (responsibly, with coordinated disclosure for any real bugs).
- **Screen recordings**: 3-minute demos of each skill, posted to YouTube and embedded in README.
- **Benchmark posts**: quarterly "how has Logic-Lens evolved?" data posts using `iteration-N/summary.json`.
- **Guest contributions**: propose case-study articles to AI/programming newsletters.

## Tracking

Open a `discussion` titled "Outreach log" where we record:
- Each marketplace submission + PR URL + status
- Each community post + link + initial response
- Monthly star-count + fork-count snapshot
- Contributor applications

This way we can see which channels drove growth and double down.

## When to declare Layer N complete

- Layer 1: all infrastructure items checked off (✅ v0.4.3)
- Layer 2: accepted into at least 2 third-party marketplaces
- Layer 3: at least one post in each of Reddit / HN / Dev.to / Twitter / Discord
- Layer 4: at least one public case-study published

## Ownership

Layer 1 and Layer 2 PR drafts can be generated by AI assistant (Claude Code).
Layer 3 posting requires the human maintainer's social accounts.
Layer 4 content drafts can be co-authored; publication is the human's call.
